### 🚀 Featured Project

**[HydroPod](https://github.com/Harshvt8585/Hydropod-)** — A portable, IoT-enabled water quality monitoring 
and purification system. Uses ESP32 + pH/TDS/Turbidity sensors + 3-stage RO filtration, 
streaming live data to a Blynk cloud dashboard.  
> Built for remote and disaster-affected communities where clean water access is unreliable.
